CSS (Cascading Style Sheets) is a style sheet language used to control the presentation and layout of HTML elements on a webpage. It enhances the visual appeal by allowing developers to define styles such as colors, fonts, spacing, and positioning for elements. CSS operates on a cascading priority system, meaning styles defined closer to the element (inline styles) take precedence over external or embedded styles.

There are three main types of CSS:
Inline CSS: Applied directly to an HTML element using the style attribute.
Internal CSS: Written within a style tag in the head section of an HTML document.
External CSS: Written in a separate file with a .css extension and linked to the HTML file using a link tag.
